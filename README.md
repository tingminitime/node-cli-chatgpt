# node-cli-chatgpt

A chatbot based on ChatGPT in terminal

> Source : [实现在终端运行的 AI 聊天机器人 ｜ ChatGPT & Nodejs](https://youtu.be/yGupa5Fq3vY)

## Features

- Support chat context
- `exit` to quit chat
- Show loading

## Technologies

- openai
- @inquirer/prompts
- dotenv
- chalk
- ora
- rollup
- typescript

## Usage

Start to chat : 

```bash
pnpm start
```

You can input `exit` to quit chat.
